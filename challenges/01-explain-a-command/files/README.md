# Challenge 1: Explain a Mystery Command

## Quick Reference

Use `gh copilot explain` to understand any shell command:

```bash
gh copilot explain "your command here"
```

## Files in this sandbox

- **mystery.sh** — A script full of complex commands to decode

## Tips

- Wrap the command in quotes when passing to `gh copilot explain`
- You can explain multi-command pipelines too
- Try `cat mystery.sh` to see all the commands first
