# Challenge 4: Debug a Broken Script

## Quick Reference

```bash
# Run the script to see errors
python3 buggy.py

# Ask Copilot to explain an error
gh copilot explain "NameError: name 'scor' is not defined"

# Ask Copilot to suggest a fix
gh copilot suggest "fix NameError in Python"
```

## Hints

There are **5 bugs** in `buggy.py`. Work through them one at a time:
1. Run the script
2. Read the error
3. Use `gh copilot explain` to understand it
4. Use `gh copilot suggest` to find the fix
5. Edit the file with `nano buggy.py` or `vi buggy.py`
6. Repeat!
